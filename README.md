# TaskManager

Бэкенд-сервис для управления задачами, разработанный на чистой Java без использования готовых фреймворков. Проект демонстрирует навыки работы с Java Core, многопоточностью, сетевым взаимодействием и архитектурой REST API.

## 🛠 Стек технологий
* **Язык:** Java Core (версии 17/21)
* **Сетевой слой:** Встроенный Java HTTP Server
* **Формат данных:** Google Gson (сериализация и десериализация JSON)
* **Тестирование:** JUnit 5
* **Сборка проекта:** Maven
* **Персистентность:** File I/O (модуль сохранения состояния приложения в файлы)

## 🚀 Что было сделано и реализовано
* **Собственный HTTP-сервер:** Настроил кастомную маршрутизацию и обработку входящих REST-запросов (GET, POST, DELETE).
* **Кастомное логирование:** С нуля спроектировал и написал систему логирования истории просмотров задач.
* **Сохранение данных:** Реализовал персистентность — приложение сохраняет свое текущее состояние в файлы и восстанавливает его при перезапуске.
* **Тестирование:** Покрыл бизнес-логику и ключевые модули приложения юнит-тестами для проверки корректности работы сервиса.
