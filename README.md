Стек: Java Core, Java HTTP Server, Gson, File I/O, JUnit, Maven.
Что сделано: С нуля разработал бэкенд-сервис управления задачами на чистой Java без использования готовых фреймворков. Реализовал собственный HTTP-сервер для обработки REST-запросов, кастомную систему логирования истории просмотров тасок и модуль сохранения состояния приложения в файлы (персистентность данных). Для сериализации данных интегрировал библиотеку Gson. Написал юнит-тесты для верификации бизнес-логики приложения.
